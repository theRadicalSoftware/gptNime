# GPTNime Quick Read

GPTNime is a local-first anime watch ledger built with Vite, React, and TypeScript. It tracks library progress, ratings, favorites, rewatches, episode memory, smart shelves, fan stats, notifications, and the anime-channel player.

Use [`README.md`](README.md) for setup, feature details, storage notes, and project structure. Use [`docs/EASTER_EGGS.md`](docs/EASTER_EGGS.md) for the subtle easter eggs and quiet UX touches.
